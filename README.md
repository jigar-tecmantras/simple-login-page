# Simple Login Page

This repository contains a lightweight, accessible login page built with plain HTML, CSS, and JavaScript.

## Running locally

1. Clone the repository: `git clone https://github.com/jigar-tecmantras/simple-login-page.git`
2. Open `index.html` in any modern browser.

The form includes inline validation and a password visibility toggle to simulate a friendly login experience without a backend.

## Project structure

- `index.html` – The markup for the login screen.
- `assets/styles.css` – Responsive styling, centered layout, and focused states.
- `assets/app.js` – Form validation logic, error messaging, and password toggle behavior.

Customize the colors or validation rules directly inside the CSS and JavaScript files.
